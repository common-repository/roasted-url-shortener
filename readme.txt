== Description ==

The plugin is for those people who don't want to use third party services to create shortened urls for their wordpress posts. The plugin simply creates another shareable link for your wordpress posts that you can share with people.


== Installation ==

Nothing fancy in here. Go to wordpress admin's plugin page, find the plugin and activate it. Oh wait. Have you copied the plugin to the plugin's directory yet?
